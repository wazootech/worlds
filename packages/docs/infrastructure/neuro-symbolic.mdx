# Neuro-Symbolic Architecture

At the heart of Worlds is a **Neuro-Symbolic Architecture**. This is our
solution to the "vibe-based" nature of modern AI.

While Large Language Models (LLMs) are exceptional at processing unstructured
language (Neural), they are notoriously prone to hallucinations and lack strict
logical structure. By marrying them with Symbolic Logic, we create a system that
is both intuitive and deterministic.

## The Problem: The Reasoning Gap

Modern AI agents often fail when they need to perform structured reasoning.
Vector search can find "similar" things, but it cannot reliably answer questions
like _"Who is the manager of the person who committed this code?"_

This is the "Reasoning Gap."

## The Solution: Neural + Symbolic

Worlds uses a hybrid model that provides the best of both worlds:

1. **Neural (Neural Intuition)**: We use generative AI to extract meaning from
   raw, unstructured data and to provide high-speed semantic search using vector
   embeddings.
2. **Symbolic (Ground Truth)**: We store that meaning as **RDF Triples** (atomic
   facts) in a queryable graph. This allows for strict logical filtering and
   verifiable reasoning using **SPARQL**.

This architecture ensures that your agent isn't just "guessing" the next token
based on patterns. Instead, it is navigating a **structured map of reality**.

By filling the **"Reasoning Gap"** with symbolic logic, we provide a perfect
audit trail for every fact the agent retrieves. The agent uses neural intuition
to find the general "area" of the knowledge, and symbolic logic to pinpoint the
exact relationship.
