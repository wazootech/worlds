The Worlds SDK provides a high-level, type-safe interface for managing worlds,
executing SPARQL queries, and performing vector searches.

## Installation

Install the SDK via JSR:

```bash
npx jsr add @wazoo/worlds-sdk
```

## Basic usage

Initialize the client with your API key.

```typescript
import { WorldsSdk } from "@wazoo/worlds-sdk";

const sdk = new WorldsSdk({
  baseUrl: "https://api.wazoo.dev",
  apiKey: "your-api-key",
});

// Run a SPARQL query
const result = await sdk.worlds.sparql(
  "WORLD_ID",
  `
  SELECT ?s ?p ?o WHERE {
    ?s ?p ?o .
  } LIMIT 10
`,
);

console.log(result);
```

## AI SDK tools

The `@wazoo/worlds-ai-sdk` package provides neuro-symbolic tools that bridge
LLMs and structured RDF knowledge. These tools enable agents to interact with
the world graph autonomously.

### Core tools

- **`discover-schema`**: Introspects a world's graph to identify available
  classes and properties. This anchors the agent in the actual vocabulary of the
  world, significantly reducing hallucinations.
- **`execute-sparql`**: Provides a safe interface for agents to execute
  structural queries. It restricts query types to ensure reasoning integrity and
  database safety.
- **`generate-iri`**: Utility for creating semantically consistent
  Internationalized Resource Identifiers (IRIs) for new entities and
  relationships.
- **`search-entities`**: Direct integration with the hybrid search engine,
  enabling agents to discover nodes via similarity or keyword matching.

### Framework integration

These tools are compatible with frameworks like the **Vercel AI SDK**, allowing
you to expose them as standard tools during model conversations.
