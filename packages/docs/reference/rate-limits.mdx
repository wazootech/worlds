---
title: "Rate Limits"
description: "Detailed policy for service account rate limiting."
---

Rate limits are enforced **per service account** using a **token bucket** algorithm. Admin requests are exempt.

### Mechanism

- **Algorithm:** Token bucket (stored in `rate_limits` table).
- **Period:** 60,000 ms (1 minute).
- **Identification:**
  - **Admin:** Exempt.
  - **Service Account:** Limited per account.
  - **Unauthenticated:** Rejected (401).
