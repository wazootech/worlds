The Worlds CLI (Command Line Interface) allows you to interact with the Worlds
engine securely from your terminal.

## Core management

- [worlds create](/reference/cli/create) - create a new isolated world
- [worlds get](/reference/cli/get) - retrieve specific world details
- [worlds list](/reference/cli/list) - list all available worlds
- [worlds update](/reference/cli/update) - modify world configuration
- [worlds delete](/reference/cli/delete) - permanently delete a world

## Data operations

- [worlds import](/reference/cli/import) - inject RDF knowledge into a world
- [worlds export](/reference/cli/export) - export a world's graph to disk

## Query & interaction

- [worlds search](/reference/cli/search) - perform semantic and keyword searches
- [worlds sparql](/reference/cli/sparql) - execute raw SPARQL queries against
  the graph
- [worlds chat](/reference/cli/chat) - interact conversationally with a world
