# Welcome to Worlds

Worlds is a **Neuro-Symbolic Memory API** designed for the AI era. It provides a
stateful infrastructure layer that bridges the gap between the probabilistic
nature of LLMs and the deterministic precision of structured data.

Our context infrastructure enables operators to build unique agents with perfect
recall and auditable reasoning by maintaining a **persistent, verifiable world
model**.

## Core Philosophy

Worlds is built on three core pillars that bridge the gap between AI intuition
and deterministic ownership.

### 1. Neuro-Symbolic Synthesis

At the heart of Worlds is a hybrid architecture designed to solve the "reasoning
gap" in modern AI.

- **Neural (LLM Intuition)**: Generative AI extracts meaning from unstructured
  data.
- **Symbolic (RDF Logic)**: That meaning is stored as deterministic facts in a
  queryable graph. This ensures your agent isn't just "guessing" context based
  on probabilities, but is navigating a **structured map of reality**.

### 2. Edge-First Autonomy

We believe an agent's memory should live as close to the user as possible.
Whether on a local machine, a private server, or at the network edge, Worlds
ensures that intelligence and knowledge live where you do.

- **Local Autonomy**: Agents never depend on a central cloud for their memory.
- **Low Latency**: Retrieving memory from a local node is significantly faster
  than a global database.

### 3. The Local-First Manifesto

As agents become more autonomous, the knowledge they carry must remain the
property of the humans they serve.

- **Residency**: Data lives close to you.
- **Ownership**: You own the triples; Worlds is the facilitator.
- **Portability**: Your graph is an open standard (RDF), not a proprietary silo.
