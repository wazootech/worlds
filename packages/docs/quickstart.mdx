## Getting Started

Follow these steps to integrate Worlds into your AI application.

### 1. Install the SDK

Choose your preferred environment to get started with the Worlds SDK.

<CodeGroup>

```bash npm
npm install @wazoo/worlds-sdk
```

```bash yarn
yarn add @wazoo/worlds-sdk
```

```bash pnpm
pnpm add @wazoo/worlds-sdk
```

</CodeGroup>

### 2. Initialize your first World

Create a connection to the Worlds API and initialize your first persistent
context.

```typescript
import { WorldsWorld } from "@wazoo/worlds-sdk";

const world = new WorldsWorld({
  apiKey: "YOUR_API_KEY",
  worldId: "my-first-world",
});

await world.init();
```

### 3. Add context

Start feeding your AI agents with persistent, searchable memory.

```typescript
await world.insert({
  text: "The secret ingredient is always more compute.",
  metadata: { source: "internal-memo" },
});
```
